# ZDESAWEB25

## Descripcion del proyecto
###### Este proyecto trata sobre una pagina web de un taller de mecanica en la cual se resuelve el problema de gestion de citas. Este proyecto ayudara a organizar mejor al taller y mejorar la comunicacion de este con los clientes.

## Publico objetivo
###### Este proyecto será derigido a cualquier persona con problemas en su coche, moto, o a aquellos que quieren modificar a su vehiculo tanto en la parte anterior con exterior.

## Funcionalidades principales 
###### Las funcionalidades principales son: mejorar la comunicacion con los cliente, facilitar al cliente crear citas, y organizar al taller.

## Tecnologías previstas
###### En este proyecto usare las siguientes herramientas:

### Frontend 
###### ***HTML***, ***CSS***, ***JAVASCRIPT***

### Backend 
###### ***PHP***, ***MARIADB***
